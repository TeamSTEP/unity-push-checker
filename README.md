# Unity Push Checker

> A GitHub App that checks for important changes made to the Unity project without any noise.

![screenshot](img/test-build.jpg)

## Setup

```sh
# Install dependencies
yarn

# Run with hot reload
yarn build:watch

# Compile and run
yarn build
yarn start
```
