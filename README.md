# Unity Push Checker

> A GitHub App built with [Probot](https://github.com/probot/probot) that checks Unity game project repositories.

## Setup

```sh
# Install dependencies
yarn

# Run with hot reload
yarn build:watch

# Compile and run
yarn build
yarn start
```
